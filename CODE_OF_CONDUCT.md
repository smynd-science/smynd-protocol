# Code of Conduct

Smynd Protocol uses a simple contribution standard:

- Be respectful.
- Assume good intent.
- Discuss technical disagreements with evidence.
- Do not publish private data.
- Do not harass, threaten, or target contributors.
- Do not use this repository to deanonymize participants or researchers.

Maintainers may remove comments, issues, or pull requests that violate these expectations.
